เอกสารเหล่านี้คือ คู่มือการเรียนการสอนวิชา ICT24467 ของมหาวิทยาลัยศรีปทุม ซึ่งเน้นการฝึกปฏิบัติจริงบนระบบ AWS Cloud ภายใต้ข้อกำหนด Free Tier เนื้อหาครอบคลุมตั้งแต่พื้นฐานการสร้างเซิร์ฟเวอร์เสมือนด้วย Amazon EC2 พร้อมการตั้งค่า Auto Scaling เพื่อรองรับภาระงานที่เพิ่มขึ้นอย่างอัตโนมัติ นอกจากนี้ยังรายละเอียดการสร้างระบบ Data Pipeline แบบ Serverless ตลอดระยะเวลา 4 สัปดาห์ โดยใช้บริการหลักอย่าง S3, Lambda, SQS, และ Step Functions เพื่อประมวลผลข้อมูลและแจ้งเตือนผ่าน SNS อย่างเป็นระบบ นักศึกษาจะได้เรียนรู้แนวคิดสำคัญทั้ง Event-Driven Architecture และการบริหารจัดการสิทธิ์ความปลอดภัยแบบ IAM เพื่อประยุกต์ใช้ในระดับมืออาชีพ สรุปบทเรียนยังรวมถึงการสร้าง CloudWatch Dashboard สำหรับตรวจสอบการทำงานและแนวทางการควบคุมค่าใช้จ่ายบนคลาวด์ให้เป็นศูนย์อีกด้วย
